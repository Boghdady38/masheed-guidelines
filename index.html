06 / USAGE PATTERNS

أنماط الاستخدام المشترك

تحدد هذه الأنماط كيفية ظهور العلامات التجارية معًا في مختلف السياقات لضمان الاتساق البصري.

PATTERN A
مشاريع السعودية

النمط A — الهيمنة الأم

تظهر مشيد كعلامة رئيسية بحجم أكبر مع ظهور العلامة الفرعية بحجم أصغر كعنصر تابع.

PATTERN B

النمط B — الظهور المتوازي

يُستخدم عند الحاجة لظهور "مشيد" كشريك رسمي متساوٍ في القوة البصرية مع العلامة التابعة.

PATTERN C

BY MASHEED

النمط C — التأييد النصي

تظهر العلامة الفرعية بشكل بارز مع إشارة نصية "By Masheed" لتأكيد الانتماء للعلامة الأم.

PATTERN D

POWERED BY

النمط D — التظهير الراعي

يُطبق على المشاريع المستقلة ذات الشخصية البصرية القوية مع عبارة "Powered by" وشعار مشيد لتأكيد الرعاية.

07 / DO'S & DON'TS

الاستخدام الصحيح والخاطئ

يجب الالتزام بقواعد الاستخدام التالية لضمان سلامة العلامة التجارية واتساقها في جميع التطبيقات.

✓ صحيح
استخدم الشعار بأبعاده الأصلية مع المساحة الآمنة المحددة
✗ خاطئ
لا تقم بتمديد أو ضغط الشعار أو تغيير أبعاده النسبية
✓ صحيح
استخدم النسخة البيضاء على الخلفيات الداكنة
✗ خاطئ
لا تضع الشعار على خلفيات مزدحمة أو ملونة بشكل يقلل الوضوح
✓ صحيح
استخدم الفاصل العمودي عند وضع علامتين بجانب بعضهما
✗ خاطئ
لا تغيّر ألوان الشعار أو تطبّق عليه فلاتر غير معتمدة
